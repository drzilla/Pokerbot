# TM91000003 — M6 3BP Wet-Turn Medium-Equity Barrel
# ============================================================
# Scenario: HU 3BP, Hero BTN opens 2.5BB with Ah Kh, BB 3bets to
# 8BB, Hero calls. 3BP flop Qs 9h 4h (single FD — hearts). Both
# check is rejected by a better test; instead Hero IP scenario:
# BB checks flop, Hero bets flop small. BB calls. Turn 8h — now
# 4-board has three hearts (FD completed from Hero's perspective
# — Hero HAS the nut FD and it came in — so Hero now has a flush;
# that's TOO strong for M6). Instead use a different turn:
#
# Re-designed: Hero has Ah Kh, flop Qs 9h 4c (one FD). Turn 8h —
# now board is Qs 9h 4c 8h → two hearts on board (4c is a club,
# so 4-board hearts = Q,9,8 = three hearts is WRONG — Qs isn't a
# heart). Recount: flop Qs 9h 4c has one heart. Turn 8h adds
# another heart → 4-board has two hearts (9h, 8h). Hero Ah Kh
# contains two hearts → Hero has a FLUSH DRAW not a made flush.
# That's medium equity = nut FD = qualifies for M6.
#
# Hero bets turn → MARGINAL per M6.
#
# Expected: mistake with type containing 'M6', confidence MARGINAL.
# Rationale: Hero barrels nut FD (medium equity in M6 terms) on a
# turn that introduces a second flush-possibility texture. Per M6,
# should check behind — a CR here forces Hero to fold and lose
# the nut-FD equity. Correct: check, realize equity for free.
# ============================================================
